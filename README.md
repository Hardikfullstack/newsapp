Local development
=================

#### step 1: first insert all API keys to new .env file shared in PDF

`REACT_APP_GUARDIANAPI=*******
 REACT_APP_NYTIMES=*******
 REACT_APP_NEWSAPI=*******`

#### step 2: install dependencies

`npm i`

#### step 3: start the App

`npm run start`

### For Docker

run below command make sure your docker is running in your system

`docker compose up`
